About the Virtual Vending Machine

The Virtual Vending Machine Webapp simulates a real-world vending experience entirely online. Users can browse, select and purchase snacks through a sleek and interactive interface — complete with a keypad, dynamic cart and personalised product recommendations.

Key Features

  • Smart Cart: Add, remove and modify items with real-time total updates.
  • Smart Recommendations: Personalized suggestions based on user preferences.
  • Authentication System: Guest and user accounts with different privileges.
  • Receipt Generator: View detailed receipts after successful transactions.
  • Keyboard & Virtual Input: Use either method to interact with the vending machine.
  • Transaction History: Easy viewing of all of your previous purchases, with filtering and sort-by settings.
  • Analytics Page: Interactive visual diagrams for analysing your purchase data

User Access

Guests can explore the vending machine and simulate purchases, but only registered users have access to proper features such as: Smart Recommendations, Saved Transaction History and User Analytics.

Technology Stack

  • Spring Boot (Java)
  • Thymeleaf Templating
  • HTML, CSS, JavaScript
  • REST APIs
  • MySQL
